GDPR Simple Notification is a lightweight plugin inform users about using cookies and about your privacy policy according to GDPR privacy policy

== Description ==

GDPR Simple Notification is a lightweight plugin that only needs to be activated and configure few inputs to make it work. Its supports GDPR privacy policy and informs user about cookies in your website.

No need to paste shortcodes, only activate the plugin.

== Installation ==

To install you must takes these steps

1. Unzip folder with plugins and upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Find in menu "Cookie notification" item and click it to configure plugin
4. After configure plugin create pages with cookie privacy and privacy policy to link them
